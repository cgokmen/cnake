Draws a single pixel in GBA Mode 4.

Note that because of the lack of single-byte edit capability on the VRAM hardware, this function ends up having to read two-pixel bundles and edit half of it. If you are setting an even number of pixels to a single color, this is not the optimal solution.

Parameters
